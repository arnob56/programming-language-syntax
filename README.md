# 🌍 Programming Language Syntax

> **A comprehensive syntax reference for 100+ programming languages.**

This repository contains concise and beginner-friendly syntax examples for a wide range of programming languages. It is designed to be a quick reference for developers, students, and anyone learning or switching between languages.

Instead of searching the internet every time you forget a syntax, simply open this repository and find the example you need.

---

# 📚 What's Included

Each language follows a consistent structure and includes examples of:

* ✅ Hello World
* ✅ Comments
* ✅ Variables
* ✅ Data Types
* ✅ User Input
* ✅ Output
* ✅ Operators
* ✅ If / Else
* ✅ Switch / Match (if supported)
* ✅ Loops (For, While, Do-While)
* ✅ Arrays / Lists
* ✅ Strings
* ✅ Functions
* ✅ Recursion
* ✅ Structs / Classes
* ✅ Objects (if supported)
* ✅ Exception Handling
* ✅ File Read
* ✅ File Write
* ✅ Basic Standard Library Usage
